Chili Change-Up

Unity · C# · Adobe Photoshop · Adobe Illustrator

Chili Change-Up is a simple puzzle game where players try to gain as many points as possible by matching 3 or more of the same chilis. The player uses mouse or touch controls to move chilis on the grid, and once 3 or more line up they are removed and points are awarded.

Game Modes

Timed — Achieve the highest score within a set time limit.

Move Limit — Achieve the highest score within a set number of moves.

Obstacle — Remove all milk cartons by making matches near them.
