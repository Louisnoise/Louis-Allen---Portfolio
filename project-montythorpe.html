Montythorpe's Big Adventure

Flash / AS3 · 26-Hour Game Jam

A physics-based adventure game developed in a 26-hour game jam. It follows the trials and tribulations of an anthropomorphic sausage as he sails to trade with new-found lands. The game was developed in Flash using AS3 at the prototyping stage, with the finished product intended for mobile devices.

Gameplay

The player controls Montythorpe's boat and must ensure the safe transportation of cargo across harsh sea conditions by controlling the balance of the boat. Players are rewarded with upgrade options based on how much cargo reaches its destination. As the game progresses, wave intensity and sea creatures create increasing obstruction.

Levels

Level 1: America

Level 2: Egypt

Level 3: China

Wave System

The waves are code-generated using a node system to appear fluid and dynamic. Each node follows a wave pattern that updates as each level progresses, creating disruption in the previous patterns and producing larger waves.

Game Structure

Controls

Controls utilise tablet gyroscopic functionality, with the player tilting the device clockwise and anti-clockwise to balance the boat.
